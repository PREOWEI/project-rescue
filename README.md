# Project Rescue

Project Rescue is an educational requirements engineering game prototype. Players analyse realistic stakeholder statements, identify valid software requirements, classify them as functional or non-functional, assign priorities, and receive feedback on their decisions.

## Features

- Project-based levels with healthcare, fintech, education, e-commerce, transport, hospitality, nonprofit, energy, and aviation scenarios
- Multiple task modes: identify requirements, classify requirements, prioritise requirements, and mixed analysis
- Score calculation based on correctness, including penalties for invalid selections
- XP and career role progression
- Portfolio tracking with best score, attempt count, completion type, and last played status
- Assisted learning mode with revealable answers
- Replay variation through shuffled statement order, wording variation, and stakeholder updates
- Timer, tips, leaderboard review, light mode, and dark mode

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open the local prototype:

```text
http://localhost:4028
```
